## Projects & Achievements

- **FaceRecg** — Facial Recognition with Deep Learning *(Completed)*

  Built a Siamese CNN (inspired by Koch et al.'s *Siamese Neural Networks for One-Shot Image Recognition*) with a shared-weight embedding network and L1 distance layer, trained on LFW for one-shot face verification — landing **93% precision and recall**. Deployed it as a real-time OpenCV webcam pipeline with tunable thresholds for live biometric authentication.

- **SusAct** — Suspicious Activity Detection in Public Places (Federated Learning) *(Completed)*

  Hit **0.81 test accuracy** training across 30 federated clients simulating real-world non-IID data, while preserving data heterogeneity through a federated learning approach — all in service of cutting communication costs without cutting corners.

- **BrnTmr** — Brain Tumor Recognition and Detection (Federated Learning) *(Completed)*

  A full computer vision pipeline for brain tumor detection, simulating a decentralized setup across 5 clients to boost model efficiency, improve generalization, and reduce communication overhead.
